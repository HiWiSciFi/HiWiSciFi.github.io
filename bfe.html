BFE

The unnecessary Brainfuck code editor, debugger and interpreter

What is Brainfuck?

Brainfuck (or Bf, Brainf****, ...) is an esoteric programming language consisting of only 8 characters, each linked to a individual functionality.

About the project

The project started as a project for school and therefor it had to be written in java and is cross-plattform. After the project had ended, some more bugfixes and improvements have been made but the program is far from perfect. The future of BFE is uncertain and it will not be continued if there is no demand to.

For executing the program you'll have to have a version of the "Java SE" installed. It can be downloaded on the official website by oracle:
https://www.oracle.com/java/technologies/javase-downloads.html

The program and source code can be found on my GitHub:
https://github.com/HiWiSciFi/BFE/releases/latest


Features

  • Interpreter
  • Memory insight
  • ASCII table
  • Breakpoints

The builtin interpreter can be restricted to move on to the next instruction in a specified time intervall. The execution can also be paused and stepped through one instruction at a time with the mere click of a button.

To make debugging a little easier, the "memory insight" feature will show you all the existing cells and their current value as hex and decimal values.

The ASCII table makes the development of Brainfuck scripts much easier beacause it enables you to look up ASCII codes for any character on the fly.

By inserting a '!' you can set a breakpoint that will pause the execution of the program once reached no matter the current execution speed.
